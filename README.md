# Problem Links Collection

This repository contains a collection of programming problem links for easy reference and tracking.

## How to Use

- Add a new line for each problem you want to track
- Use this format for each entry:
  ```
  - [Problem Title](URL) — Platform: *Platform Name* — Difficulty: *Level*
  ```

## Examples

- [Two Sum](https://leetcode.com/problems/two-sum) — Platform: LeetCode — Difficulty: Easy

- [Binary Search](https://www.hackerrank.com/challenges/binary-search) — Platform: HackerRank — Difficulty: Medium
